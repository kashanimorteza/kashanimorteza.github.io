## Install
#### Windows
	Download [atom files] and copy to [C:\atom\core]
	Create folder [.atom] into [C:\atom]
	setx PATH %path%;C:\atom\core
	setx ATOM_HOME C:\atom\.atom
#### Linux
	sudo dpkg -i atom-amd64.deb
