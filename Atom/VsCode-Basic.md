## Install

#### Windows

	Download [VsCode files] and copy to [C:\VsCode\core]
	Create folder [.VsCode] into [C:\VsCode]
	setx PATH %path%;C:\VsCode\core
	setx VsCode_HOME C:\VsCode\.VsCode

#### Linux
	sudo dpkg -i VsCode-amd64.deb
